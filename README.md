# TZF Viewer for Android

Цель проекта — автономно открывать и отображать файлы Trimble TZF на Android.
Приложение не использует RealWorks и не требует предварительной конвертации в XYZ.

Сейчас реализован каркас Android-приложения и нативный модуль `tzfreader`.
Нативный модуль читает и проверяет заголовок TZF; декодер блоков точек добавляется
по результатам сверки с установленным RealWorks на тестовых файлах.

Для сборки нужен Android SDK с NDK и JDK 17:

```powershell
./gradlew assembleDebug
```
