# TZF point-cloud preview APK

## Цель

APK автономно открывает Trimble `.tzf` через системный выбор файлов и показывает интерактивный предпросмотр облака точек. Windows DLL RealWorks и предварительный экспорт не используются.

## Архитектура

`MainActivity` получает `content://` URI и передаёт доступный file descriptor в NDK. Нативный слой читает TZF, извлекает сферические координаты из компонентов 1–3, переводит их в XYZ и формирует ограниченную выборку точек. Выборка возвращается в Java как прямой `ByteBuffer` с `float x,y,z`.

`GLSurfaceView` загружает этот буфер в VBO и рендерит его через OpenGL ES 3.0 как `GL_POINTS`. Камера центрируется по границам облака; один палец вращает, жест двумя пальцами меняет масштаб.

## Ограничение памяти и время отклика

Декодер не обязан держать весь скан в памяти. Он формирует равномерную выборку до заданного безопасного предела точек, начиная с 150 000. Этого достаточно для предпросмотра миллионов измерений на типичном Android-устройстве.

## Ошибки

Во время чтения показывается прогресс. Повреждённый TZF, неподдержанный кодек или ошибка доступа к URI приводят к понятному сообщению на русском. Пустой предпросмотр не считается успешным результатом.

## Проверка

Нативные unit-тесты сохраняются. APK собирается в GitHub Actions. Для реального TZF проверяется, что декодер возвращает ненулевое количество XYZ-точек и что OpenGL-экран получает буфер и границы облака.
